# eslint-config-ibood

> ESLint shareable config to be used with XO


## Install

```
$ yarn add --dev xo eslint-config-paprika
```

## Usage

Add some XO config to your package.json:

```json
{
	"name": "my-new-project",
	"xo": {
		"extends": ["ibood"]
	}
}
```

Or to .eslintrc:

```json
{
	"extends": ["xo", "ibood"]
}
```
